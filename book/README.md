# Python for Finance

This folder contains notebooks that follow along Yves Hilpisch's *Python for Finance*, which is published by O'Reily and can be purchased [here](https://www.oreilly.com/library/view/python-for-finance/9781492024323/). I recommend you visit [Yves'](https://github.com/yhilpisch) platform, the [*Python Quant Platform*](https://quant-platform.com/Python_Quant_Platform.html) to learn more about the source material. His code is copyrighted and the book is only allowed to be reproduced for personal use, so my notebooks are not published to this repository. I advocate you purchase his book and follow along with the excellent material.
