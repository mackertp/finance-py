# finance_py
Follows Yves Hilpisch's Python for Finance Book

This repository follows Yves Hilpisch's *Python for Finance*, which is published by O'Reily and can be purchased [here](https://www.oreilly.com/library/view/python-for-finance/9781492024323/). I did not create the original code, but I have added to some of the notebooks and made a few changes to ensure the code compiles within my environment. I recommend you visit [Yves'](https://github.com/yhilpisch) platform, the [*Python Quant Platform*](https://quant-platform.com/Python_Quant_Platform.html) to learn more about the source material, but happy to answer any questions. 

If you'd like to follow along with my notes, I've included my conda environment setup with a yaml file. To get started, simply run: 

```console
uname@os:~$ conda env create -f finance_py.yml
```
